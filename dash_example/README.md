# Dash Example

This is an example application demonstrating how to use the Dash admin panel framework.

## Features Demonstrated

- Basic Panel configuration
- Resource registration (Users, Posts)
- Navigation groups and sorting

## Running the Example

```bash
dart run lib/main.dart
```

## Structure

```
lib/
├── main.dart              # Main entry point
└── resources/
    ├── user_resource.dart # User resource example
    └── post_resource.dart # Post resource example
```
